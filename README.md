# My Python Practice

This repository contains practice scripts for learning Python basics and intermediate topics. The code demonstrates arithmetic operations, control flow, functions, closures, decorators, and simple games.

## Project structure

- `variables.py`: Basic variable manipulation, type conversions, and user input conditions.
- `binary.py`: Binary/decimal conversion examples and binary arithmetic.
- `Calculator.py`: A simple command-line calculator supporting +, -, *, /.
- `game.py`: Number guessing game (1 to 20) with user input loop.
- `intermediate_functions.py`: Intermediate concepts:
  - decorators (`repeat`)
  - closures (`make_multiplier`)
  - higher-order functions (`apply_operation`)
  - `*args`/`**kwargs` usage
  - type hints and a discount calculator
  - lambdas, map/filter
  - function composition
- `print.py`: (if present) likely includes print-related practice usage and output examples.
